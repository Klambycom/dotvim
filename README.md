My .vim
=======

Installation
------------

Download:

	git clone git://github.com/klambycom/dotvim.git

Install plugins:

	cd ~/.vim
	git submodule init
	git submodule update

Create symlink:

	ln -s ~/.vim/vimrc ~/.vimrc

Adding Plugin Bundles
---------------------

	cd ~/.vim
	git submodule add https://github.com/tpope/vim-markdown.git bundle/markdown
